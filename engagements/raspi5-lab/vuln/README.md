# Phase 3: Vulnerability Analysis

## What Goes Here
- CVE lookups for discovered services
- Vulnerability scanner output
- Manual testing results
- ExploitDB search results
- Proof of concept notes
- Risk assessment

## Common Commands
```bash
# Search for known exploits
searchsploit <service> <version>

# Check CVE databases
# Use web search or vulners.com

# Nmap vuln scripts
nmap --script vuln <target>

# Manual testing notes go here as markdown
```

## Checklist
- [ ] Each discovered service checked for known CVEs
- [ ] ExploitDB searched
- [ ] Vulnerability scanner run (if appropriate)
- [ ] Manual testing performed
- [ ] Findings documented with severity
