# Phase 6: Reporting

## What Goes Here
- Executive summary
- Technical findings
- Risk ratings (Critical/High/Medium/Low/Info)
- Evidence references
- Remediation recommendations
- Methodology description
- Scope confirmation

## Report Structure
1. **Executive Summary** — Non-technical overview for management
2. **Scope & Methodology** — What was tested and how
3. **Findings** — Each vulnerability as a separate section
   - Description
   - Impact
   - Evidence (screenshots, output)
   - Remediation
4. **Appendices** — Raw scan output, tool versions, etc.

## Checklist
- [ ] Executive summary written
- [ ] All findings documented
- [ ] Evidence attached
- [ ] Remediation advice provided
- [ ] Report reviewed for accuracy
- [ ] Sensitive data handled properly
