## Found: Confirmed Vulnerabilities / Observations

### 1. RDP TLS 1.0/1.1 Support Enabled — Configuration Weakness
- **Finding:** RDP service supports legacy TLS versions (1.0, 1.1) alongside TLS 1.2 and 1.3
- **Evidence:** Nmap ssl-enum-ciphers script confirmed TLSv1.0 and TLSv1.1 cipher suites enabled
- **Risk:** Legacy TLS versions have known weaknesses (BEAST, POODLE variants). While patched Windows systems mitigate these, disabling legacy versions is recommended.
- **CVSS:** N/A (configuration issue, not CVE)
- **Remediation:** Disable TLS 1.0/1.1 via registry or Group Policy; enforce TLS 1.2 minimum
- **Confidence:** ✅ **High** — Directly observed
