## Attack Surface Summary

### Primary Vectors

1. **MySQL (3306/tcp)** - HIGH INTEREST
   - Known version 8.4.8 - check for CVEs
   - Default auth plugin may allow auth bypass attempts
   - Weak/default credentials likely if dev environment

2. **SMB (445/tcp)** - MEDIUM INTEREST
   - Windows SMB - check for NULL sessions, shares, vulnerable versions
   - Could reveal files, user lists, OS patches via MS17-010 et al

3. **RDP (3389/tcp)** - MEDIUM INTEREST
   - NLA enabled (good security posture)
   - CredSSP - potential for CVE-2018-0886 (Oracle encryption) check
   - Brute force risk if weak credentials

4. **RPC (135/tcp, 49153/tcp)** - LOW-MEDIUM
   - DCOM/RPC attack surface (PetitPotam, PrintNightmare prerequisites)
   - Requires authenticated access for most vectors

### Filtered Ports

94 ports filtered suggests Windows Firewall or third-party firewall active. Default-deny posture detected.
