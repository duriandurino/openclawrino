# Reporting Fields for Vulnerability Findings

Minimum fields:

```text
Title
Affected Asset
Validation Status
Evidence
Applicable CVE(s)
Relevant CWE(s)
CVSS
Exploitability Notes
Business Context
Remediation
Hardening
Retest Guidance
```

## Rule
Do not present a candidate CVE as a confirmed finding unless applicability and evidence are both established.
